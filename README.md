# AWS Examples Lab Setup

Este repositorio está configurado para trabajar con el AWS CLI.

## Uso local (VS Code)

### Instalación AWS CLI
Windows: instalar MSI oficial  
WSL/Linux:
```bash
sudo apt-get update
sudo apt-get install unzip -y
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
